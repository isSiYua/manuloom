# Reference outputs

Examples in this directory are designed to demonstrate ManuLoom's public output contract without redistributing a creator's transcript, screenshots, or article images.

| Example | Source | Purpose |
|---|---|---|
| [Synthetic pipeline note](synthetic-pipeline-note/manuloom-demo.md) | Original repository-owned script and diagrams | Demonstrates information density, structure, visual placement, tables, code, attribution, and limitations |

Real-source examples may be added only when their text and images have a compatible license or the repository has explicit publication permission. A generated note is not automatically safe to redistribute merely because the source URL is public.
